## Basic MDX Syntax

MDX supports all standard Markdown syntax plus JSX components. Here are some examples:

### Standard Markdown Elements

- **Bold text** and *italic text*
- `Inline code` and code blocks
- [Links](https://astro.build) and images
- Lists and blockquotes

> This is a blockquote that demonstrates standard Markdown functionality working alongside JSX components.

### Custom Components

You can import and use any React component in your MDX files:

```jsx
import { Button } from "@/components/ui/button";

<Button>Click me!</Button>
```

## Code Examples with Syntax Highlighting

MDX preserves all markdown features, including syntax-highlighted code blocks:

```typescript
// Example: Creating a custom MDX component
interface Props {
  title: string;
  children: React.ReactNode;
}

export function Callout({ title, children }: Props) {
  return (
    <div className="my-6 p-4 border-l-4 border-blue-500 bg-blue-50 dark:bg-blue-950/20">
      <h4 className="font-semibold text-blue-900 dark:text-blue-100">
        {title}
      </h4>
      <div className="mt-2 text-blue-800 dark:text-blue-200">
        {children}
      </div>
    </div>
  );
}
```
